# 概要设计文档

## 1. 产品概述

### 1.1 核心目标
将教学知识点快速转换为**教学视频**和**教学PPT**

### 1.2 用户故事
```
作为一名教师，
我输入一个知识点（如"牛顿第二定律"），
系统自动生成对应的教学PPT和讲解视频，
让我无需手动制作即可获得可用的教学资源。
```

## 3. 核心流程详细设计

### 3.1 课程生成流程
```
Step 1: 知识点输入
────────────────────
输入: "牛顿第二定律"
选择: 学段(初中/高中)、学科(物理)

Step 2: 课程大纲生成
────────────────────
输入: 知识点 + 学段学科
LLM分析 → 输出课程章节结构
输出示例:
{
  "chapters": [
    {"title": "定律引入", "duration": "5min", "key_points": ["生活实例"]},
    {"title": "定律表述", "duration": "10min", "key_points": ["公式", "单位"]},
    {"title": "公式推导", "duration": "15min", "key_points": ["推导过程"]},
    {"title": "例题讲解", "duration": "20min", "key_points": ["例题1", "例题2"]}
  ]
}

Step 3: 讲稿脚本生成
────────────────────
输入: 课程大纲
LLM生成 → 每章节对应的讲解脚本
输出: 完整的文字讲稿
```

### 3.2 PPT生成流程
```
Step 1: PPT结构生成
────────────────────
输入: 课程大纲
AI分析 → PPT页面结构
输出: 
{
  "slides": [
    {"type": "title", "content": "牛顿第二定律"},
    {"type": "content", "content": "F=ma", "notes": "重点讲解"},
    {"type": "image", "content": "受力分析图"},
    {"type": "quiz", "content": "随堂练习"}
  ]
}

Step 2: PPT内容填充
────────────────────
调用PPT生成服务，填充具体内容
支持的生成方案:
- PptxGenJS: 代码生成PPT
- Marp: Markdown转PPT
- 模板填充: 基于预设模板

Step 3: PPT渲染导出
────────────────────
输出: .pptx 文件
```

### 3.3 视频生成流程
```
Step 1: 数字人配置
────────────────────
选择: 数字人形象、声音
配置: 语速、音量

Step 2: 配音生成 (TTS)
────────────────────
输入: 讲稿脚本
TTS服务 → 生成音频文件
输出: .wav/.mp3 音频

Step 3: 视频合成
────────────────────
输入: 音频 + PPT画面/数字人口型
视频合成服务 → 输出视频文件
支持的合成方案:
- 数字人口播 + PPT画面
- 纯数字人讲解
- PPT配音解说

Step 4: 视频输出
────────────────────
输出: .mp4 视频文件
```

## 4. 主要接口设计

### 4.1 课程生成接口
```json
POST /api/course/generate
Request:
{
  "knowledge_point": "牛顿第二定律",
  "subject": "物理",
  "grade": "高中",
  "chapters_count": 4
}

Response:
{
  "course_id": "course_xxx",
  "outline": {...},
  "script": "..."
}
```

### 4.2 PPT生成接口
```json
POST /api/ppt/generate
Request:
{
  "course_id": "course_xxx",
  "template": "default",
  "style": "简洁"
}

Response:
{
  "ppt_id": "ppt_xxx",
  "status": "generating"
}
```

### 4.3 视频生成接口
```json
POST /api/video/generate
Request:
{
  "course_id": "course_xxx",
  "avatar_id": "avatar_xxx",
  "voice_id": "voice_xxx",
  "has_avatar": true,
  "has_ppt_background": true
}

Response:
{
  "video_id": "video_xxx",
  "status": "generating"
}
```

## 7. 用户操作流程

### 7.1 快速生成流程（推荐）
```
1. 输入知识点
      ↓
2. 点击"一键生成"
      ↓
3. 等待生成（可查看进度）
      ↓
4. 预览PPT和视频
      ↓
5. 下载或分享
```

### 7.2 分步生成流程（可选）
```
1. 输入知识点
      ↓
2. 生成课程大纲 → 查看/调整
      ↓
3. 生成讲稿脚本 → 查看/调整
      ↓
4. 生成PPT → 查看/调整
      ↓
5. 生成视频 → 查看/下载
```

## 8. 非功能性需求

### 8.1 性能要求
- 课程大纲生成：< 10秒
- PPT生成：< 30秒
- 视频生成：< 5分钟（3分钟视频）

### 8.2 生成质量
- PPT结构清晰、符合教学规范
- 视频画面流畅、口型同步
- 讲稿内容准确、表达清晰

### 8.3 资源格式
| 产物 | 格式 | 说明 |
|------|------|------|
| PPT | .pptx | 兼容Office/WPS |
| 视频 | .mp4 | H.264编码 |
| 讲稿 | .md/.txt | 可复制编辑 |
| 教案 | .docx/.md | 可编辑 |
