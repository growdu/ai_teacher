# AI Teacher Studio

智能教学资源生成平台 — 将知识点快速转换为教学视频、PPT、随堂测验等多元教学资源。

## 核心能力

<div class="grid cards" markdown>

-   :material-clock-fast: **快速生成** — 输入知识点，5分钟内生成完整课程资源
-   :material-brain: **AI 驱动** — 大模型赋能，内容准确、形式生动
-   :material-account-group: **互动教学** — 内置师生互动节点，实时反馈学习效果
-   :material-book-open-variant: **多元资源** — PPT / 视频 / 测验一站式生成

</div>
